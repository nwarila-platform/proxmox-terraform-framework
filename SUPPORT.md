# Support

This portfolio framework is maintained as a public reference implementation for
Proxmox cluster Terraform patterns.

For usage questions, open a regular issue with:

- the Terraform version from `terraform/versions.tf`
- the Proxmox provider version from `terraform/versions.tf`
- the command that failed
- the smallest relevant configuration snippet

Do not include API tokens, state files, plan files, host credentials, or private
network inventory in an issue.
